# Task API

Simple REST API for managing tasks (Node.js + Express + MongoDB).

## Features
- Create task (title required) with validation
- List tasks
- Get task by id
- Delete task by id
- consistent error responses
- Uses MongoDB + Mongoose

## Requirements
- Node 18+ recommended
- MongoDB (local or Atlas)
- npm
